EST-iMs System
ABOUT US

EST-iMs is a professional inventory management system built specifically for Alestesharia Animal Nutrition — delivering real-time stock control across all warehouse zones.

The Company
Alestesharia Animal Nutrition is a leading company in the animal feed sector, managing multiple warehouse zones with diverse inventory categories.
The System
EST-iMs (Inventory Management System) is a web-based platform built on Flask & Python, using Excel files as structured data storage with real-time read/write capabilities.
Warehouse Zones
The system manages multiple zones: Zone 1–5 for warehouse operations, QC for quality control, and Admin/Dev for management and development access.
Access Control
Role-based access control ensures each employee only accesses their assigned zone. Super-zones (Admin, Dev) have full visibility across all warehouse zones.
Key Features
Real-time stock tracking with IN/OUT transaction logging
QR Code scanning for quick inventory lookups from any mobile device
Excel-based data storage with automated balance recalculation
Dark / Light mode with full Arabic & English language support
CSV export, print reports, and an AI assistant for system guidance
Brute-force protection, session management, and secure authentication
